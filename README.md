# invoice-OCR-extraction-system
## Overview

The Invoice OCR Extraction System is a desktop application that extracts text and structured data from invoices using Optical Character Recognition (OCR). It helps automate invoice processing and reduces manual data entry.

## Features
- Extracts key invoice details: Invoice Number, Date, Vendor, Amount, Line Items
- Supports multiple invoice formats
- Generates structured output (CSV/JSON)
- User-friendly interface with one-click extraction
  
## Installation
- Download the latest release from the Releases page.
- Extract the ZIP if needed.
- Run main.exe to start the application.
  
## Usage
- Open main.exe.
- Click “Load Invoice” and select your invoice file.
- Click “Extract Data”.
- Export results as CSV or JSON.
